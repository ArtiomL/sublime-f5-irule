## Description

Sublime Text package for F5 iRules syntax highlighting and auto-completion.

## Installation

macOS:

```
cd ~/Library/Application\ Support/Sublime\ Text\ 3/Packages/
git clone https://github.com/ArtiomL/sublime-f5-irules.git
```
